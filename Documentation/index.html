The nairn-mpm-fea Project

Introduction

This information is about the nairn-mpm-fea computational mechanics project.

Code Engines

NairnMPM and NairnFEA Code Engines

The open-source calculation engines NairnMPM and NairnFEA do material point method (MPM) and finite element analysis (FEA) calculations, respectively. They are object-oriented, C++, platform-independent, parallel code engines.

OSParticulas Code Engine

The calculation engine OSParticulas is a development version of this code project for the most current developments and for features that are not yet available in the open-source code. Those eligible for using OSParticulas will be given instructions on obtaining it and on its extra features.

Documentation

For complete documentation on the nairn-mpm-fea computational mechanics project, the user is referred to the web-based wiki called OSUPDocs. For times you are working on code, but do not have internet access, the following topics give reference documentation that may be enough to use the code.

  • Organization - organization of the folders in this project.
  • Getting Starting - compiling the source code.
  • Running the Code - running the code engines.
  • Input File Format - explains how to get documentation for the commands needed to set up FEA or MPM calculations.
  • Visualization - options for visualizing the results (on line only).
  • Output Results Format - format for output files of FEA and MPM results (on line only).

Technical Notes
