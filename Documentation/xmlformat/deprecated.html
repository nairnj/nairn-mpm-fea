Deprecated Commands

The following commands have been deprecated. The most common reasons are to change the default units for the command or to change the style of the input. These commands all still work.

<TimeStep>(time step)/>
Replaced by <Timing>. Enter (time step) for the MPM time step in sec (or determined by a units attribute). Deprecated in 11.0.0.
<MaxTime>(max time)/>
Replaced by <Timing>. Enter (max time) for the time that the calculations will stop in sec (or determined by a units attribute). Deprecated in 11.0.0.
<TimeFactor>(time factor)/>
Replaced by <Timing>. Enter (time factor) for the Courant-Friedrichs-Levy (or CFL) factor to use when determining the MPM time step. The time step used in the analysis will be the minimum of the one determined using this factor and the one entered in (time step). The default CFL is 0.5. Deprecated in 11.0.0.
<ArchiveTime maxProps="(props)">(time)</ArchiveTime>
Replaced by <ArchiveGroup>. Enter (time) for the time interval between archiving in sec (or determined by a units attribute). The (props) attribute forces archiving if (props) propagations and/or debonds have occurred since the last archive (default is 0, which disables counting of propagations). Deprecated in 11.0.0.
<FirstArchiveTime>(first time)</FirstArchiveTime>
Replaced by <ArchiveGroup>. Enter (first time) for the first time results are archived in sec (or determined by a units attribute). Deprecated in 11.0.0.
