Temperature

When using explicity defined meshes, the temperature of each node can be set in the explicit <NodeList> section. When the mesh is defined by commands that generate the mesh, you can set temperature and/or thermal gradients using the <Thermal> element as follows: 

<Thermal>
  <Temperature>20</Temperature>
  <StressFreeTemp>25</StressFreeTemp>
</Thermal>

where the elements are:

<Temperature>
A user-supplied function to define the temperature field. The function will be evaluated at each node and the resulting temperature will be applied to that node. A temperature field will result in residual thermal stresses.
<StressFreeTemp>
This tag allows you to specify a stress free or reference temperature. It is only the difference between Temperature and StressFreeTemp that matters.

Some more information is on the OSUPDocs wiki.
