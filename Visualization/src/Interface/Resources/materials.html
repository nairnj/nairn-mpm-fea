Material Types

NairnFEA and NairnMPM support a variety of material types. Details on each material are given on the OSUPDocs wiki for MPM Materials, for FEA Materials, and for traction law materials; this section just lists the available materials and the properties needed to use them in calculations.

A material or traction law is defined in scripting commands using a block that allows you to specify a new material type and define all its properties. The command block format is:

Material (id),(name),(type)
  (Property commands - one per line)
Done

where (id) is the ID or Label for the material created by the command, (name) is the material name entered as a string, and (type) is the material type, which is enter by material name or number in the table below.

The Material command should be followed by lines that contain the material properties. Each property should be on a separate line and when the last property is specified, enter a line with the single Done Command. To set a property, use the property name (case sensitive) followed by a space and then its value. For example, to set modulus of an isotropic material, use

E 2300

A few properties differ from the above (name,value) pair; those exceptions are in the material definitions.

Linear Viscoelastic, Isotropic (Type=6)

This material is a small-strain, linear viscoelastic material with time-dependent bulk and shear moduli given by sums relaxation times. The pressure law can use small strain methods (with time-dependent bulk modulus) or large-deformation Mie-Grüneisen equation of state (MGEOS) (with time-independent, non-linear elastic bulk modulus). The input properties are:

  • G0 - the steady-state (or long time) shear modulus (in pressure units)
  • Gk - the shear modulus for the next relaxation time (in pressure units)
  • tauk - the next shear modulus relaxation time (in time units)
  • pressureLaw - set to 0 to use small strain method (with provided K terms) or 1 to use MGEOS (with all its properties). MGEOS cannot be used with time-dependent bulk modulus.
  • K - the steady-state (or long time) bulk modulus (in pressure units) (when pressureLaw=0)
  • Kk - the bulk modulus for the next relaxation time (in pressure units) (when pressureLaw=0)
  • tauKk - the next bulk modulus relaxation time (in time units) (when pressureLaw=0)
  • MGEOS Properties - for pressureLaw=1, set C0, S1, S2, S3, gamma0, and Kmax for the MGEOS calculations. The UJOption is fixed at 1.
  • Tref - reference temperature to shift relaxation times using the WLF equation. If Tref<0, then no shifting is done and relaxation times will be independent of temperature.
  • C1 - coefficient in WLF equation used when Tref>0 to shift relaxation times. If the entered value is negative, the thermal shift switches to Arrhenious activation energy with ΔHa = -RC1.
  • C2 - coefficient in WLF equation used when Tref>0 to shift relaxation times (not used if C1 is negative).
  • mref - reference concentration to shift relaxation times using the WLF-style equation. If mref<0, then no shifting is done and relaxation times will be independent of concentration.
  • Cm1 - coefficient in WLF-style equation used when mref≥0 to shift relaxation times.
  • Cm2 - coefficient in WLF-style equation used when mref≥0 to shift relaxation times/
  • bTemp
    bTValue - enter (bTemp,bTValue) pairs for piecewise interpolation of vertical temperature shift of elastic properties. Enter any number of pairs with bTemp values monotonically increasing.
  • bConc
    bCValue - enter (bConc,bCValue) pairs for piecewise interpolation of vertical concentration (or moisture) shift of elastic properties. Enter any number of pairs with bConc values monotonically increasing.
  • alpha - the thermal expansion coefficient (in ppm/K) (ignored for MGEOS when pressureLaw=1)
  • (other) - see the properties common to all materials (MPM only).

The relaxation shear modulus is given by G(t) = G0 + G1exp(-t/tau1) + G2exp(-t/tau2) + ... + Gntausexp(-t/taun), where ntaus is the number of shear modulus relation times entered (and that number must exactly match the number of Gk shear moduli entered).

The relaxation bulk modulus is given by K(t) = K + K1exp(-t/tau1) + K2exp(-t/tau2) + ... + KntausKexp(-t/taun), where ntausK is the number of bulk modulus relation times entered (and that number must exactly match the number of Kk shear moduli entered). To use time-independent bulk modulus, enter only K.

For more details relaxation time and elastic property shifting see the OSUPDocs wiki.

This material can be used in plane stress analysis, but only if pressureLaw=0 and artificial visocity is not used.

History Data for Linear Viscoelastic, Isotropic Materials

Mooney-Rivlin Hyperelastic, Isotropic (Type=8)

The material type is a large-deformation, isotropic, hyperelastic material. The material properties are

  • G1 or G2 - the Mooney-Rivlin shear energy depends on two shear moduli (in pressure units). Their sum in the low-strain shear modulus. If G2=0 (which is its default), the material is a Neohookean material.
  • K - the K modulus (in pressure units), which is equal to the low-strain bulk modulus.
  • E an nu - alternative enter E (in pressure units) and nu. These are converted to K and G=G1+G2; you can pick G2 (if not zero) to get G1=G-G2
  • UJOption - selects the dilational energy term U(J) and the possible integer values mean:
        0: U(J) = (K/2)((1/2)(J2-1) - ln J) (the default)
        1: U(J) = (K/2)(J-1)2
        2: U(J) = (K/2)(ln J)2
  • alpha - the thermal expansion coefficient (in ppm/K)
  • IdealRubber - this property should by 0 or 1 where 1 makes the material an ideal rubber elastic material.
  • (other) - see the properties common to all materials (MPM only).

History Data for Mooney-Rivlin Materials

  1. The volumetric ratio or J = V/V0 (i.e., the determinant of the deformation gradient)

The plastic strain stores the left Cauchy Green tensor.

Clamped Neo-Hookean Material (Type=29)

The material type is a large-deformation, isotropic, hyperelastic material. It yields when the tensile and/or compression elongation reaches critical values of 1+θs and/or 1-θc. The underlying elastic component of the deformation is a neo-Hookean material. After yielding, the mechanical properties soften in tension or harden in compression according to an entered hardening coefficient. The material properties are

  • CritTens and CritComp - enter critical strains θs and θc for critical strains for tension and compression. If either is less than zero, clamping is prevented and the material will use the choosen hyperelastic, neo-Hookean model (see "Elastic" property).
  • xihard - hardening parameter ξ. The neo-Hookean moduli are scaled by exp(ξ(1-JP)) where JP is the plastic volume ratio.
  • Elastic - enter 0 to use a modified, corotated, neo-Hookean potential energy defined in paper that proposed this material or enter 1 to use Neo-Hookean material potential energy. This potential energy applies to elastic component of the deformation.
  • (other) - enter properties for the underlying Neo-Hookean material (note that the UJOption can only be changed when Elastic is 1; UJOption is always 1 when Elastic is 0).

History Data for Clamped Neo-Hookean Material

  1. The volumetric ratio or J = V/V0 (i.e., the determinant of the deformation gradient)
  2. The plastic volumetric ratio or JP = J/JE where JE is determinant of the elastic deformation gradient)

The plastic strain stores the left Cauchy Green tensor. You have to archive both strain and plastic strain for the visualization to be able to plot elastic and plastic strains.

Bistable, Isotropic Elastic (Type=10)

This material has two small-strain, isotopic states. You need to enter properties of the initial and deformed states and pick rule for state transformations:

  • K0 and Kd - bulk modulus of initial state and transformed state (in pressure units).
  • G0 and Gd - shear modulus of initial state and transformed state (in pressure units).
  • alpha0 and alphad - thermal expansion coefficient of initial state and transformed state (in ppm/K).
  • beta0 and betad - concentration expansion coefficient of initial state and transformed state (in strain/(wt fraction)).
  • D0 and Dd - diffusion constant of initial state and transformed state (in diffusion units).
  • kCond0 and kCondd - thermal conductivity of initial state and transformed state (in conductivity units).
  • transition - set state transition to be by a dilation rule (if 1), a distortion rule (if 2), or Von Mises stress rule (if 3) (the default is dilation).
  • critical - critical value for the transition. The value is volumetric or deviatoric strain (in % for dilation or distortion rule) or stress (in pressure units) for Von Mises stress rule.
  • DeltaVOffset - offset volumetric strain in the transformed state. This property only applies for dilation rule (see above); it is ignored for distortion rule.
  • reversible - set to "yes" (or 1) to make the transition reversible and "no" (or 0) to make it irreversible (the default is reversible).
  • (other) - see the properties common to all materials (MPM only).

Rigid Material (Type=11)

The material is actually two materials in one, depending on the value for its SetDirection property. If SetDirection is 0 through 7, the material is not really a material. Instead material points with such a material will assign grid-based boundary conditions that translate with the particle. If SetDirection is 8, the material will correspond to actual material points that interact with other material points only through multimaterial contact laws. The material properties are:

  • direction - values 0 through 7 (binary 0 through 0x111) create dynamic velocity boundary conditions in the x, y, and/or z axis according to three bits in the binary value (e.g.,1 or binary 0x001 to set velocity in the x direction). A setting of 8 creates rigid particles that can implement multimaterial contact contact.
  • temperature - set to 1 to create dynamic temperature boundary conditions (it requires direction 0 to 7).
  • concentration - set to 1 to create dynamic concentration boundary conditions (it requires direction 0 to 7).
  • SettingFunction, SettingFunction2, or SettingFunction3 - by providing user-defined functions of time, position, and time step, the rigid particle velocity setting can vary. The functions should evaluate to the velocity in velocity units. For SetDirection 1 to 7 the functions must be given in order and apply to the set velocities in order. For SetDirection of 8, the three functions are for setting x, y, and z directions and zero to three can be used.
  • SettingFunctionX, SettingFunctionY, or SettingFunctionY - synonyms for SettingFunction, SettingFunction2, or SettingFunction3, respectively.
  • ValueFunction - by entering user-defined functions of time, time step, and position, the rigid particle temperature and concentration can vary. The function should evaluate to the temperature (in K) or concentration potential.
  • mirrored should be set to -1 or +1 to indicate that rigid particles with SetDirection = 1 to 7 are setting a velocity on the minimum edge (-1) or maximum edge (+1) of the object along the moving direction. This property can improve the performance of velocity boundary conditions as explained on the OSUPdocs wiki. The default setting of 0 means to not use mirroring. Note that when ExtrapolateRigid option is activated, this property is ignored.

Hill Plastic, Orthotropic Material (Type=15)

This material is a small-strain, elastic-plastic material based on an orthotropic material but has plasticity determined by a built-in, anisotropic criterion for yielding:

[ F(σyy - σzz)2 + G(σxx - σzz)2 + H(σyy - σxx)2 + 2Lτyz2 + 2Mτxz2 + 2Nτxy2 ]1/2 - H(α)

where H(α) is hardening law as function of cumulative plastic strain and F through N are determined from yield stress in each material direction. The material properties are:

  • yldxx - yield stress from uniaxial tension test the material's x direction (in pressure units)
  • yldyy - yield stress from uniaxial tension test the material's y direction (in pressure units)
  • yldzz - yield stress from uniaxial tension test the material's z direction (in pressure units)
  • yldxy - yield stress from pure shear test in the material's x-y plane (in pressure units)
  • yldyz - yield stress from pure shear test in the material's y-z plane (in pressure units)
  • yldxz - yield stress from pure shear test in the material's x-z plane (in pressure units)
  • khard - hardening term (see below)
  • nhard - exponent on cumulative plastic strain in hardening term (see below)
  • exphard - coefficient in exponential hardening law (see below).
  • alphaMax - optional α where exponential hardening converts to linear hardening.
  • HillStyle - use 1 for square-root yield criterion or 2 for squared yield criterion (default is 2). See the OSUPDocs wiki for why two options are available.
  • (other) - see Linear Elastic, Orthotropic material for all other properties.

The τyz and τxz terms and their yield stresses are not needed for 2D analyses.

If nhard>0, the hardning law is H(α) = 1 + Khard*αnhard, but if nhard<0, the hardening law is changed to H(α) = (1 + Khard*α)nhard. If exphard is provided instead, the hardening law is exponential or H(α) = 1 + (Khard/exphard)*(1-exp(-exphard*α)). If both nhard and exphard are provided, the last one provided will determine the hardening law. If alphaMax is provided exponential hardening converts to linear softening using the final slope of the exponential hardening law (this parameter is used to prevent exponential hardening from decaying to elastic-plastic response).

For more details see the OSUPDocs wiki.

History Data for Hill Plastic, Orthotropic Material

The plastic strain will have the plastic strain. You have to archive both strain and plastic strain for the visualization to be able to plot elastic and plastic strains.

Mie-Grüneisen Equation of State Materials (Type=25 or Type=17)

This material is isotropic with plasticity and find pressure using the Mie-Grüneisen equation of state by large deformation methods. It is a large-deformation, hyperelastic-plastic methods for shear and yielding can use any hardening law for plasticity. Note: the Type=17 material used to be a small-strain material, but is now deleted and both types now use large deformation methods. The material properties are:

  • C0 - the bulk wave speed (in alt velocity units).
  • S1 - a parameter in Mie-Grüneisen equation of state (dimensionless).
  • S2 - a parameter in Mie-Grüneisen equation of state (dimensionless).
  • S3 - a parameter in Mie-Grüneisen equation of state (dimensionless).
  • gamma0 - a parameter in Mie-Grüneisen equation of state (dimensionless).
  • Kmax - maximum increase allowed in bulk modulus (by ratio and dimensionless) (default is -1 or no max).
  • G - the shear modulus at zero pressure and the reference temperature (in pressure units).
  • UJOption - Use 0, 1, or 2 to select energy function to use for finding pressure when in tension (see Mooney material) (default 0).
  • Hardening - this property selects the hardening law. It should be followed by entering all yielding properties required by the hardening law within the material's definition block.
  • (other) - see the properties common to all materials. This material requires heat capacity Cv and a StressFreeTemp in K.

In this material, the pressure is related to the volumetric compression strain (x = -ΔV/V0) by the Mie-Grüneisen equation of state. When x>0, the material is in compression and pressure is

P = ρ0C02x(1 - 0.5 γ0 x) / (1 - S1x - S2x2 - S3x3)2 + ρ0γ0e

where e is the internal energy. When x<0, the material is in tension and the pressure changes to

P = ρ0C02x + ρ0γ0e

The first term uses the low-strain bulk modulus, which is ρ0C02.

This equation of state automatically handles thermal expansion (assuming a valid Cv was provided) and therefore you do not need to enter a thermal expansion coefficient.

History Data for Mie-Grüneisen Equation of State Material

All hardening law will have one or more history variables. This material then tracks relative volume (J = V/V0) in the first history variable after hardening law history variables.

The plastic strain stores the elastic, left Cauchy Green tensor. You have to archive both strain and plastic strain for the visualization to be able to plot elastic and plastic strains.

Tait Liquid (Type=27)

This material models a Newtonian fluid as a hyperelastic material. The pressure is found from the Tait equation. The properties are:

  • K - the zero-pressure, bulk modulus (in pressure units)
  • viscosity - liquid viscosity at one shear rate (in viscosity units)
  • logshearrate - log of shear rate corresponding to a viscosity (in 1/time units).
  • alpha - the thermal expansion coefficient (in ppm/K)
  • InitialPressure - set initial pressure to user-defined function of position that evaluates to a pressure in pressure units. When liquids are modeled in gravity, the function can set to ρgh, where h is height of liquid above the position (x, y, z).
  • (other) - see the properties common to all materials (MPM only).

Triangular Traction Law Material (Type=12)

The traction law has a triangular shape. There are separate traction laws for opening displacement (mode I) and sliding displacements (mode II):

  • sigmaI and sigmaII - the peak stress or cohesive stress of the traction law (σ) (in pressure units).
  • JIc and JIIc - the toughness of the traction law or area under the traction law curve (Jc) (in energy release units)
  • delIc and delIIc - the maximum or critical opening displacement (δc) at which debonding occurs (in length units)
  • delpkI and delpkII - the relative displacement at the location of the peak in the triangular law (relative to δc and therefore dimensionless from 0 to 1).
  • kIe and kIIe - the initial elastic slope (k) (in pressure units/length units).
  • nmix - the exponent used in the mixed-mode failure criterion Enter nmix<=0 to choose infinity, which is decoupled failure criterion (see below).

You must then enter exactly two of the first three properties for each mode (because they are interrelated). You must enter zero or one for the next two properties. If one is provided, the other will be calculated. If neither k nor δpeak are entered, the peak will be located at δpeak = 0.225926299δc.

Failure occurs when

(GI/JIc)nmix + (GII/JIIc)nmix = 1

where GI and GII are areas under mode I and mode II traction laws up to current displacements. If nmix<=0, failure occurs when either mode reaches its critical COD.

This material tracks two history variabeles:

  1. Maximum normal opening displacement (or equal to normal δe prior to initiation).
  2. Maximum shear opening displacement (or equal to tangential δe prior to initiation)

This traction law assumes a linear elastic response up to σc followed by a scaled exponential decrease to reach zero at critical crack opening displacement (COD) of δc.

Notice how the shape changes with α. In the limit of α=0, this law is identical to the Triangular Traction Law (and that one should be used instead because it is more efficient). Coulomb Friction with Adhesion Contact Law (Type=63)

This law implements Coulomb friction law with adhesion (in OSParticulus only). If the surface are in contact, the frictional sliding force is

Sslide = Sa + (µ +kΔv)N

where Sa is the tangential adhesion strength and µ is the coefficient of friction, and k allows the dynamic coefficient of friction to depend linearly on sliding velocity. The µ term models transition from static to dynamic coefficient of friction by

µ = (µs vhalf + µd v)/(vhalf + v)

where vhalf is velocity for which coefficient of friction is half way from static coefficient of friction (µs) to dynamic coefficient of friction (µd). The default is vhalf=0, which corresponds to instantaneous drop from µs to µd at the onset of sliding.

If the surfaces are not in contact, the surface continue to stick as long as:

(Sstick/Sa)2 + (N/Na)2 < 1

where Na is the normal adhesion strength. If this criterion is not met (or if either Sa or Na are zero), the surfaces move freely with zero tractions. The properties for this law are:

  • coeff for the dynamic coefficient of friction (default is 0).
  • coeffStatic for the static coefficient of friction (default is -1).
  • Sa for tangential adhesion strength (in pressure units) (default is 0).
  • Na for normal adhesion strength (in pressure units) (default is 0).
  • kmu - slope of coefficient of friction vs. velocity (in 1/velocity units) (default is 0).
  • vhalf - velocity at which µ is half way from µs to µd (in velocity units) (default is 0).
  • displacementOnly 0 for contact by separation and stress, 1 for contact by separation only, or <0 for contact by sepaction and interfacial stress < -displacementOnly (in pressure units/length units) (default is 0).
  • Dc <0 to find separation assuming perfect interfaciall contact, ≥0 to use imperfect interface methods to find separation (in pressure units/length units) (default is -1)

If the optional static coefficient of friction is changed to a positive number, it must be greater than the dynamic coefficient. The sliding will begin when the sliding force overcomes the frictional force calculated from static coefficient and v=0, but thereafter will slide with force found using the velocity dependent dynamic coefficient of friction.

The damage initiates when either the maximum principle stress exceeds the tensile strength or the maximum shear stress exceeds the shear strength. This law couples failure when needed and allows pressure dependent shear strength. For more details see the OSUPDocs Wiki. The input properties are:

  • sigmac - the tensile strength (in pressure units). The default is very large number which means the material will not fail in tension.
  • tauc - the shear strength (in pressure units). The default is very large number which means the material will not fail in shear.
  • PressureModel - pick a model (0 to 4) to model pressure dependent shear strength (default is zero for pressure independent).
  • tauh - hydrostatic tension that causes shear strength to drop to zero for pressure model 1 or maximum shear strength for pressure models 2, 3, and 4 (in pressure units). The default is infinite.
  • P1, P2 - break points for shear strength in pressure models 2, 3, and 4 in pressure units). The default is 0 for P1.
  • maxPressure - the maximum pressure expected in a simulations. It is used to verify damage mechanics stability at the start of the simulation. |in pressure units). The default is 0.

Transversely Isotropic Failure Initiation (number=2)

The damage initiates when axial stress exceeds axial tensile strength, axial shear stress exceeds axial shear strength, principle stress in the transverse plane exceeds transverse shear strength, or maximum shear stress in transverse plane exceeds the shear strength.

  • sigmacA - critical axial strength (inMPa) for failure in the axial direction
  • sigmac - critical transverse tensile strength (in pressure units) for tensile failure in the isotropic plane
  • taucA - critical shear strength (in pressure units) for failure due to axial shear stress with failure parallel to the axial direction
  • taucT - Critical shear strength (in pressure units) for failure due to axial shear stress with failure through the axial direction
  • tauc - critical transverse shear strength (in pressure units) for shear failure in the isotropic plane

All strength values default to a large number, which means failure will not occur in that mode.

Linear Softening (number=1)

The linear softening law is f(δ) = 1 - δ/δmax. The input property is:

  • Gc - the toughness of the material is this law's failure mode (in energy release units). The default is very large number which means the material will not soften after initiation of damage.

Note that you do not enter δmax. It is determined at run time because it is affected by particle volume, initiation stress, and crack orientation.

Exponential Softening (number=2)

The exponential softening law is f(δ) = exp(-kδ). The input properties are:

  • Gc - the toughness of the material is this law's failure mode (in energy release units). The default is very large number which means the material will not soften after initiation of damage.
  • min - minimum fraction of remaining strength before particle is marked as failed (default: 0.01)

Note that you do not enter k. It is determined at run time because it is affected by particle volume, initiation stress, and crack orientation.

Cubic Step Function Softening (number=3)

This softening law is a cubic function that changes slope after initiation and then decays to failure. If the initial slope after failure is zero, it is a cubic step function given by f(δ) = 1 + (δ/δmax)2(2(δ/δmax)-3.). Other forms are explained on OSUPDocs Wiki. The input properties are:

  • Gc - the toughness of the material is this law's failure mode (in energy release units). The default is very large number which means the material will not soften after initiation of damage.
  • k - initial slope after initiation (dimensionless and must be >0) (default: 0)

Note that you do not enter δmax. It is determined at run time because it is affected by particle volume, initiation stress, and crack orientation.

Two-Exponential Softening Law (number=4)

This two-exponential softening law can model a peak and decay after initiation or just double exponential decay. Details of the function are explained on OSUPDocs Wiki. The input properties are:

  • Gc - the toughness of the material is this law's failure mode (in energy release units). The default is very large number which means the material will not soften after initiation of damage.
  • alpha - relative decay rate of the second exponential (must be greater than 1)
  • beta - relative magnitude of the the second exponential (must be less than 1)
  • min - minimum fraction of remaining strength before particle is marked as failed (default: 0.01)

Material Properties Common For All Material Types

There are some material properties that are common to all material types, although not all materials will use them all. The categories of common properties are:

  • Basic Properties
  • Fracture Toughness Properties
  • Crack Propagation Properties
  • Contact Properties
  • Artificial Viscosity
  • Poroelasticity Properties

Basic Properties

These are basic material properties

  • rho - the density (in density units)
  • Cv - the constant strain heat capacity (in heat capacity units). You do not need to enter a constant pressure heat capacity (Cp) because it is not needed. Some materials will output Cp by using thermodynamic relations for the difference between Cp and Cv, but Cp is never needed in calculations.
  • kCond - the thermal conductivity for isotropic materials (in conductivity units)
  • csat - the saturation concentration entered as a weight fraction
  • beta - the concentration expansion coefficient for isotropic materials (in strain/(wt fraction))
  • D - the diffusion constant for isotropic materials (in diffusion units)
  • largeRotation - to use (1) or not use (0, the default) polar decomposition when evaluating rotations. This option is only allowed for linear elastic and elastic-plastic small strain materials. Option 1 uses polar decomposition to evaluate incremental rotation and uses that rotation to update strains and stresses. Option 0 use approximations of polar decomposition to rotate strain and stress updates (standard hypoelasticity). Both methods work for large rotations (provided the rotations are incrementally small). Option 1 may be more accurate while options 0 is more efficient.
  • matDamping - sets custom particle damping to apply only to particles of this material. Its value replaces global particle damping setting and must be a constant (function of time not allowed).
  • matPIC - sets custom PIC damping to apply only to particles of this material. Its value replaces the global PIC damping setting. Note that to use custom XPIC damping in materials, the global settings must activate XPIC and then the matPIC property can change a material to use a different amount of XPIC or switch to FLIP. If the global settings are 100% FLIP, the matPIC property can implement PIC, but not XPIC.
  • color (red),(green),(blue),(alpha) - sets the color of the material with the four arguments being RGB and alpha values between 0.0 and 1.0. A single argument can be used instead to set gray level between 0.0 and 1.0 (with alpha=1.0). Three arguments set red, green, and blue with alpha=1.0. If no color is provided, a color will be picked from the current spectrum using the material number.

Fracture Toughness Properties

These properties set material properties that determine the fracture toughness of the material and control various aspects of crack propagation. Note that whether or not the properties are used depends on the selected criterion for crack growth.

  • JIc or JIc - critical mode I or mode II energy release rate (in energy release units).
  • KIc or KIIc - critical mode I or mode II stress intensity factor (in stress intensity units).
  • KIexp or KIIexp - the exponent p or q in the elliptical criteria for crack growth.
  • delIc or delIIc - critical crack opening displacements in mode I and mode II.
  • initTime - time to start crack growth (in alt time units).
  • speed - define the constant crack speed (in velocity units).
  • maxLength - define the maximum length of the crack (in length units). The crack growth will stop when this length is reached.
  • nmix - a parameter used in some mixed-mode failure laws.
  • allowsCracks is set 1 to allow the material to have cracks or 0 to not allow cracks. The default is 1 for non-rigid materials and 0 for rigid materials. Setting a non-rigid material to 0 lets that material enter a crack in another material such as to simulate cutting, to wedge open a crack, or model hydraulic fracturing. The non-cracking material can open the crack in the other material by contact on its crack surfaces.
  • gamma, p, or gain - still in development and not meant for general use.

Crack Propagation Properties

The criterion, direction, and traction properties (and the analogous alternate propagation properties) are set with following XML commands (instead of standard property commands):

  • criterion - set crack growth criterion for this material (see options in Propagate Command).
  • direction - set optional crack growth direction for this material (see options in Propagate Command).
  • traction - set optional traction law to create for crack propagation in this material by ID or number (see Propagate Command for details).
  • altcriterion is identical to criterion, but applies instead to the alternate crack growth criterion in the AltPropagate Command.
  • altdirection is identical to direction, but applies instead to the alternate crack growth direction in the AltPropagate Command.
  • alttraction is identical to traction, but applies instead to the alternate crack growth traction law material in the AltPropagate Command.
  • xGrow and yGrow - specify a unit vector for a constant crack growth direction. They only apply for crack propagation with criterion 2 and then only when that criterion selects the default propagation direction.
  • constantTip - set to 0 or 1 where 1 means the crack tip will always uses its initial material even if the crack is in or moves to another material

Contact Properties

These commands allow custom contact properties for each material pair in multimaterial mode MPM.

  • Contact - a Contact material property can define custom contact law for multimaterial mode MPM contact between the current material and another material.
  • shareMatField (matID) - set to another material (by material ID) to share material velocity field with that material. Two (or more) materials sharing the same velocity field will move with perfect contact (i.e., in a single velocity field) among themselves, but can interact with other materials in different velocity fields by contact or interface laws. Materials sharing velocity fields must be compatible (e.g., all rigid or all nonrigid) and the material in (matID) must not be sharing it's field.

The deprecated material properties Friction and Interface should be replaced by the above Contact property.

Artificial Viscosity

Artificial viscosity adds a pressure, Q, related to velocity gradient on the particle, but only when it is compressing. The equation is:

Q = Δx*|Dkk|*(A1*C + A2*Δx*|Dkk|)

where Δx is the cell size of the mesh, |Dkk| is the relative volume change rate, C is the bulk wave speed in the material, and A1 and A2 are adjustable constants.

  • ArtificialVisc - set to "on" or "off" to determine if artificial viscosity is on.
  • avA1 - the A1 parameter in artificial viscosity (dimensionless, default 0.2)
  • avA2 - the A2 parameter in artificial viscosity (dimensionless, default 2.0)

The artificial viscosity option is only supported in some materials (as documented here). If you use these commands in a material that does not support, an error will result.

Poroelasticity Properties

Some materials support poroelasticity calculations and the properties in this section control pore pressure flow between particles and coupling between stress and strain and pore pressure. The properties to use depend on symmetry of the parent material.

Isotropic Poroelasticity Properties

  • Ku - undrained bulk modulus (pressure units)
  • alphaPE - The poroelasticity Biot coefficient (from 0 to 1) (dimensionless, default 0)
  • Darcy - Darcy law permittivity for the material (1/length units2, default 0)

Transversely Isotropic Poroelasticity Properties

  • Ku - undrained bulk modulus (pressure units)
  • alphaAPE, alphaTPE - The axial and transverse poroelasticity Biot coefficients (from 0 to 1) (dimensionless, default 0)
  • DarcyA, DarcyT - Darcy axial and transverse law permittivities for the material (1/length units2, default 0)

Orthotropic Poroelasticity Properties

  • Ku - undrained bulk modulus (pressure units)
  • alphaxPE, alphayPE, alphazPE - The poroelasticity Biot coefficients in the three orthrotopic directions (from 0 to 1) (dimensionless, default 0)
  • alphaRPE, alphaZPE, alphaTPE - For cylindrical, orthotropic materials, these are radial, axial, and hoop Biot coefficients. They are synonyms for orthotropic coefficients (with R=x, Z=y, and T=z). (from 0 to 1) (dimensionless, default 0)
  • Darcyx, Darcyy, Darcyz - Darcy law permittivities for the three orthotropic directions (1/length units2, default 0)
  • DarcyR, DarcyZ, DarcyT - For cylindrical, orthotropic materials, these are radial, axial, and hoop Darcy law permittivities for the material. They are synonyms for orthotropic coefficients (with R=x, Z=y, and T=z). (1/length units2, default 0)
