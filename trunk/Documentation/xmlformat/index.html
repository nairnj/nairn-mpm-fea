XML Input Files

Finite element calculations using NairnFEA or Material point method simulations using NairnMPM are controlled by input from an XML input file. This input file can be created manually by using any text or XML editing software or created automatically by interpreting scripting commands in NairnFEAMPM (Mac only) or in NairnFEAMPMViz (a multi-platform Java application).

The main documentation for learning how to create the input files is in the on-line OSUPDocs wiki. This section provides a brief reference to all commands that are allowed in input files for NairnFEA or NairnMPM calculations. This brief reference is not meant for learning the codes (that is best done on the OSUPDocs wiki). But, if you have become familiar with the codes, the reference should provide enoguh information to run calculations when you done have on-line access to the OSUPDocs wiki.

Sections of NairnFEA XML Input Files

The detailed help for running NairnFEA is available here; you should refer to this documentation when you have on-line access. If you current do not have on-line access, the following topics provide brief reference for all the possible sections allowed in input XML files for running NairnFEA:

  1. File Start
  2. Header
  3. Defining the FEA Mesh
  4. Defining Materials
  5. Boundary Conditions
  6. Temperature (optional)
  7. End the File

When input files are validated using the DTD file (as recommended), the above sections must appear in the above specified order. Optional sections that are not needed may be omitted. A sample NairnFEA input file (on line) shows a sample calculation of a cantilever beam.

Sections of NairnMPM XML Input Files

The following are all the possible sections for input XML files for running NairnMPM. When input files are validated using the DTD file (as recommended), the sections must appear in the specified order. Optional sections that are not needed may be omitted. A sample NairnMPM input file shows a sample calculation of two colliding disks.

  1. File Start
  2. Header
  3. MPM Header
  4. Defining the MPM Background Grid
  5. Defining the Material Points
  6. Defining Cracks (optional)
  7. Defining Materials
  8. Grid Boundary Conditions (optional)
  9. Particle Boundary Conditions (optional)
  10. Thermal Conduction and Loading (optional)
  11. Adding Gravity Field (optional)
  12. Custom Tasks (optional)
  13. End the File
