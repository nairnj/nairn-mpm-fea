Obtaining MPM and FEA Source Code

The source code for the NairnMPM and NairnFEA calculation engines and for the NairnFEAMPMViz java application are publically available. As of December 2009, the code has be commited to a googlecode project called nairn-mpm-fea. You can obtain code from this project one of two ways:

  • Anonymous checkout for anyone
  • Secured check with ability to make changes

Anonymous Checkout

The googlecode project is publically available for anyone to checkout anonymously. First, you need a work station with svn or the subversion system used by googlecode. Versions of svn are available for Mac, Windows, and Linux. Once you have svn installed, you check out the code as follows: 

svn checkout http://nairn-mpm-fea.googlecode.com/svn/trunk/ nairn-mpm-fea-read-only

Here nairn-mpm-fea-read-only is the name of the folder that will be created with the checked out code. You can use any name you want.

See source code files for information on the contents of the checked out code. The above checkout out will get the current version of the code. See version history to read about other versions and to learn how to get those versions instead.

Secured Checkout

If you might want to contribute changes to the nairn-mpm-fea project, including changes to documentation or the project's wiki pages, or if you just want a visible presence in use of the code, you should checkout with a username and password. But first, you need a password. The process is:

  1. Participating in a googlecode project requires a google or gmail account. If you do not have one, you can create one for free at www.google.com. If you already have a google account, you can easily add an email address.
  2. Email you google email address to John.Nairn@oregonstate.edu with a request to the a committer (ability to make changes) or a contributer (to be added as a visible user of the code). You will be emailed back when you have been added to the project
  3. Sign in to your google account and find your profile in the google code section. The "My Profile" tab will show you user name and the "Settings" tab will show the GoogleCode.com password. Make of note of the password, because it is needed for checkout
  4. Finally, you need svn installed on your computer. Versions of svn are available for Mac, Windows, and Linux.

Once the above steps are down (and they only need to be done once), you can check out the code with:

svn checkout https://nairn-mpm-fea.googlecode.com/svn/trunk/ nairn-mpm-fea --username YOURUSERNAME

Here nairn-mpm-fea is the name of the folder that will be created with the checked out code. You can use any name you want. Also YOURUSERNAME is the username you obtained in setp #3 above. After giving this command, you will need the password from step #3 to finish the checkout

See source code files for information on the contents of the checked out code. The above checkout out will get the current version of the code. See version history to read about other versions and to learn how to get those versions instead.
