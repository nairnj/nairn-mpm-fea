Obtaining MPM and FEA Source Code

The source code for the NairnMPM and NairnFEA calculation engines and for the NairnFEAMPMViz java application are publically available, but your method for downloading depends on your location and your computer system:

  • User with an account on the Carbon/Hedera Cluster:
    • Checkout on the cluster
    • External checkout to work station
  • Downloads for all others

Obtaining Source Code on the Carbon/Hedera Cluster

If you have an account on the Carbon/Hedera Cluster (and membership in the nairnj group) in the Oregon State University, Wood Science & Engineering Department you can log on to the cluster and obtain the source code by cvs checkout. A typical bash session is as follows: 

mkdir mycopy
cd mycopy
export CVSROOT='/home/nairnj/Repository/'
cvs checkout NairnCode

The first command makes a new directory to hold the source code; the second command moves into that directory. The export defines a variable that tells cvs the location of the source repository. Finally, the last command copies all NairnMPM, NairnFEA, and NairnFEAMPMViz source code to your directory.

See source code files for information on the contents of the checked out code. The above checkout out will get the current version of the code. See version history to read about other versions and to learn how to get those versions instead.

Obtaining Source Code on Unix or Linux-Compatible Work Station

If you have an account on the Carbon/Hedera Cluster (and membership in the nairnj group) in the Oregon State University, Wood Science & Engineering Department, you can alternatively check out the code to any Unix or Linux-compatible work station (such as a MacOS X computer or Windows computer with cygwin and cvs installed) by using password access to cvs checkout on the cluster. A typical bash session is as follows: 

mkdir mycopy
cd mycopy
export CVS_RSH=ssh
export CVSROOT=':ext:username@carbon.fsl.orst.edu:/home/nairnj/Repository/'
cvs checkout NairnCode

The first command makes a new directory to hold the source code; the second command moves into that directory. The first export command sets connection method (and may not be needed). The second export command defines a variable that tells cvs that the repository is on a particular external server. You should enter your user name for username. Finally, the last command copies all MPM and FEA source code to your directory. Before the checkout occurs, you will have to enter your password.

See source code files for information on the contents of the checked out code. The above checkout out will get the current version of the code. See version history to read about other versions and to learn how to get those versions instead.
