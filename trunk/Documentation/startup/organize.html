NairnCode Source Code Organization

The recommended organization is to put all source code into a separate folder. Assuming tah folder's name is NairnCode, the NairnMPM, NairnFEA, and NairnFEAMPM source code is organized into various directories. The contents of the directories are:

NairnCode/Common
This folder has items common to both MPM and FEA code
NairnCode/Projects
This folder has project files for editing and compiling the code.
NairnCode/(others)
All other folders in Common have various common sections of code.
NairnCode/NairnMPM
This folder is the root folder for the NairnMPM source code
NairnCode/NairnMPM/src
This folder has all MPM-specific source code in various subfolders
NairnCode/NairnMPM/build
This folder has the makefile for compiling the code using a make command.
NairnCode/NairnMPM/input
This folder has sample input files and the NairnMPM.dtd file. The compiled code gets installed here ready for running with the input files.
NairnCode/NairnMPM/info
This folder has various text files with additional information.
NairnCode/NairnMPM/tools
This folder has source code for command line tools that can be used with NairnMPM such as for extracting data for archive files.
NairnCode/NairnFEA
This folder is the root folder for the NairnFEA source code
NairnCode/NairnFEA/src
This folder has all FEA-specific source code in various subfolders
NairnCode/NairnFEA/build
This folder has the makefile for compiling the code using a make command.
NairnCode/NairnFEA/input
This folder has sample input files and the NairnFEA.dtd file. The compiled code gets installed here ready for running with the input files.
NairnCode/NairnFEA/info
This folder has various text files with additional information.
NairnCode/Visualization
This folder is the root folder for the platform-independent, Java-based visualiztion tool.
NairnCode/Visualization/src
This folder has the Java source code for the visualization tool called NairnFEAMPMViz.
NairnCode/Visualization/info
This folder has a makejar script that can compile the Java source code into a jar file. It also has information on the visualization tool and resource files (images help text, and packages (as jar files)) needed to compile the Java application into a jar file.
NairnCode/Visualization/Projects
This folder has project files for editing and compiling the visualization tool.
NairnCode/Documentation
This folder has all documentation of the current version. Open the index.html file to view the documentation in any browser.
