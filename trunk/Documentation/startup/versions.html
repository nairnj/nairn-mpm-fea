Version History

See the NairnFEA/info/FEAVersionHistory.txt or the NairnMPM/info/VersionHistory.txt files in the source code for a brief description of the changes between NairnFEA and NairnMPM versions. This page has information on obtaining specific versions The available versions were restarted in 2006 when the source code was moved to a new Repository

Checking Out the Code

If you are obtaining the code by cvs checkout, you can specify a version using 

cvs checkout -r NairnCodevvv NairnCode

where the tag following the -r option is the desired version and vvv is the version number. If you are obtaining the code by download, you can select from the available versions in the download table. The following is a list of available versions in the repository:

cvs checkout NairnCode (current version)

A checkout with no version tag will get the most recent development version. See info/VersionHistory.txt for possible information on recently added features.

cvs checkout -r NairnCode1_5_0 NairnCode

  • Committed 14 MAR 208
  • Contains NairnFEA 2.5, NairnMPM 6.5, and NairnFEAMPMViz 2.0

cvs checkout -r NairnCode1_4_0 NairnCode

  • Contains NairnFEA 2.4, NairnMPM 6.4, and NairnFEAMPMViz 1.0
