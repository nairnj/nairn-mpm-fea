Nairn-MPM-FEA: NairnFEA v2.6.x, NairnMPM v6.6.x,
   NairnFEAMPMViz v2.1, and Documentation

This documentation is for Nairn-MPM-FEA package. The calculation engines NairnFEA and NairnMPM do finite element analysis (FEA) and material point method (MPM) calculations. The java application NairnFEAMPMViz provides platform-independent tools for running and visualizing calculations. For Mac-specific calculations and visualization (with more features than NairnFEAMPMViz), see the NairnFEAMPM web site.

This documentation explains how to obtain the source code, compile all tools, create input files for calculations, run calculations, and gives the options for visualizing and interpreting the results. The code can be run on multiple platforms including MacOS X, Windows, and Linux machines.

Documentation

The following topics describe use of the NairnCode system:

  • Start Up - download and compile all code.
  • Running the Code - run the executable code engines.
  • Visualization - options for visualizing the results.
  • XML Input File Format - definition of the XML file commands needed to set up FEA or MPM calculations.
  • Output Results Format - format for output files of FEA and MPM results
