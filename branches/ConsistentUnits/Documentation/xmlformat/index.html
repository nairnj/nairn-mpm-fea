XML Input Files

Finite element calculations using NairnFEA or Material point method simulations using NairnMPM are controlled by input from an XML input file. This input file can be created manually by using any text or XML editing software or created automatically by interpreting scripting commands in NairnFEAMPM (Mac only) or in NairnFEAMPMViz (a multi-platform Java application).

The main documentation for learning how to create the input files is in the on-line OSUPDocs wiki. This section provides a brief reference to all commands that are allowed in input files for NairnFEA or NairnMPM calculations. This brief reference is not meant for learning the codes (that is best done on the OSUPDocs wiki). But, if you are familiar with the codes, theis reference should provide enoguh information to run calculations when you do not have on-line access to the OSUPDocs wiki.

Sections of NairnFEA XML Input Files

The following topics provide brief reference for all the possible sections allowed in input XML files for running NairnFEA. When input files are validated using the DTD file (as recommended), the above sections must appear in the above specified order. Optional sections that are not needed may be omitted.

  1. File Start
  2. Header
  3. Defining the FEA Mesh
  4. Defining Materials
  5. Boundary Conditions
  6. Temperature (optional)
  7. End the File

When input files are validated using the DTD file (as recommended), the above sections must appear in the above specified order. Optional sections that are not needed may be omitted.

Sections of NairnMPM XML Input Files

The following are all the possible sections for input XML files for running NairnMPM. When input files are validated using the DTD file (as recommended), the sections must appear in the specified order. Optional sections that are not needed may be omitted.

  1. File Start
  2. Header
  3. MPM Header
  4. Defining the MPM Background Grid
  5. Defining the Material Points
  6. Defining Cracks (optional)
  7. Defining Materials
  8. Grid Boundary Conditions (optional)
  9. Particle Boundary Conditions (optional)
  10. Thermal Conduction and Loading (optional)
  11. Adding Gravity Field (optional)
  12. Custom Tasks (optional)
  13. End the File
  14. Deprecated Commands
