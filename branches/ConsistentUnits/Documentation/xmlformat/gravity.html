Gravity

MPM simulations can include gravity or any other body force (for more details see OSUPDocs wiki). Use the following structure to specify a vector in the direction of gravity and/or a body force: 

<Gravity>
  <BodyXForce>0</BodyXForce>
  <BodyYForce>-9800</BodyYForce>
  <BodyZForce>0</BodyZForce>
  <GridBodyXForce>3*x</GridBodyXForce>
  <GridBodyYForce>8*t-100*t</GridBodyYForce>
  <GridBodyZForce>12*z</GridBodyZForce>
</Gravity>

The Body(XYZ)Force values must be numeric constants. The GridBody(XYZ)Force commands can be user-defined functions of position and time. All body forces are in acceleration units. Any omitted forces are set to zero. If two forces along the same axis are given (a constant and a function), the two values are summed to get total body force.
