Defining Materials

NairnFEA and NairnMPM support a variety of material types. Details on each material are given on the OSUPDocs wiki for MPM Materials, for FEA Materials, and for traction law materials; this section just lists the available materials and the properties needed to use them in calculations.

A material is defined using a Material block:

<Material Type='1' Name='Polymer'>
     <prop>(value)</prop>
         ⋮
</Material>

where Type is the material type (integer) which defines the type of material and Name is any user-defined description of the the material (<200 characters and must be unique among defined materials if you reference them by name). The material properties for each type are defined by subordinate <prop> tags to set one property to (value).

Referencing Materials

When materials are assigned to FEA mesh elements, MPM materials points, MPM crack tips, or in a few other commands, they can be referenced by number or by name:

By Numbers
In this method, you use mat="#" attributes where # is the material number. The materials are defined by numbers in the order they appear in the output file with the first material being number 1.
By Name
In this method, you use matname="A Name" attributes where "A Name" is a unique material name that matches the name of any defined material in the file.

All defined materials will appear in the output file in the ordered referenced by name rather than in the order defined in the input file. But, if you do not reference any materials by name, they will appear in the order found in the input file. It is best to reference materials by one method in all commands (i.e., you should not reference some materials by number and others by name).

Linear Elastic, Isotropic (Type=1)

This material is small-strain, isotropic material. The material properties are:

  • E - modulus (in MPa)
  • nu - Poisson's ratio
  • alpha - thermal expansion coefficient (in ppm/C)
  • (other) - see the properties common to all materials (MPM only).

Linear Elastic, Transversely Isotropic 1 (Type=2)

This material is small-strain, transversely isotropic with the isotropic plane in the x-y plane. The properties are:

  • EA or ET - axial or transverse modulus (in MPa)
  • GA - axial shear modulus (in MPa)
  • nuA or nuT - axial or transverse Poisson's ratio
  • alphaA or alphaT - axial or transverse thermal expansion coefficient (in ppm/C)
  • betaA or betaT - axial or transverse concentration expansion coefficient (in strain/wt fraction) (MPM only)
  • DA or DT - axial or transverse diffusion constant (in mm2/sec) (MPM only)
  • kCondA or kCondT - axial or transverse thermal conductivity (in W/(m-K)) (MPM only)
  • (other) - see the properties common to all materials (MPM only).

Linear Elastic, Transversely Isotropic 2 (Type=3)

This material is small-strain, transversely isotropic with the axial direction of the material in the y direction of the x-y plane analysis. This direction can be rotated by using input angle attributes when assigning material properties to material points or to FEA elements. The input properties are identical to the previous transversely isotropic material.

Linear Elastic, Orthotropic (Type=4)

The material is small-strain, orthotropic with the input axes corresponding to the analysis x-y-z axes. The material properties can be changed by rotation about the axes:

  • Ex, Ey, or ET - x, y, or z direction modulus (in MPa)
  • Gxy, Gxz, or Gyz -x-y, x-z, or y-z plane shear modulus (in MPa)
  • nuxy, nuyx, nuxz, nuzx, nuyz, or nuzy - various Poisson's ratios (enter one of each pair with matching axes)
  • alphaz, alphay, or alphaz - x, y, or z direction thermal expansion coefficient (in ppm/C)
  • betaz, betay, or or betaz - x, y, or z direction concentration expansion coefficient (in strain/wt fraction) (MPM only)
  • Dx, Dy, or Dz - x, y, or z direction diffusion constant (in mm2/sec) (MPM only)
  • kCondx, kCOndy, or kCondz - x, y, or z direction thermal conductivity (in W/(m-K)) (MPM only)
  • (other) - see the properties common to all materials (MPM only).

Imperfect Interface Material (Type=5)

This material is used to model imperfect interfaces in FEA. The input properties are

  • Dn - stiffness for loading the interface in the normal direction (in MPa/mm)
  • Dt - stiffness for loading the interface in the tangential direction (in MPa/mm)

Linear Viscoelastic, Isotropic (Type=6)

This material is a small-strain, linear viscoelastic material with time-dependent shear modulus given by a sum or relaxation times and a time-independent bulk modulus. The input properties are:

  • K - the time-independent bulk modulus (in MPa)
  • G0 - the steady-state (or long time) shear modulus (in MPa)
  • alpha - the thermal expansion coefficient (in ppm/C)
  • ntaus - the number of relaxation times in the shear modulus. This number must exactly match the number of <Gk> and <tauk> settings and come before any of those settings
  • Gk - the shear modulus for the next relaxation time (in MPa)
  • tauk - the next relaxation time (in sec)
  • (other) - see the properties common to all materials.

The relaxation shear modulus is given by G(t) = G0 + G1exp(-t/tau1) + G2exp(-t/tau2) + ... + Gntausexp(-t/tauntaus).

History Data for Linear Viscoelastic, Isotropic Materials

This material tracks history for implementation of linear viscoelastic properties, but currently no history variables are available for archiving.

Mooney-Rivlin Hyperelastic, Isotropic (Type=8)

The material type is a large-deformation, isotropic, hyperelastic material. The material properties are

  • G1 or G2 - the Mooney-Riviln shear energy depends on two shear moduli (in MPa). Their sum in the low-strain shear modulus. If G2=0 (which is its default), the material is a Neohookean material.
  • K - the K modulus (in MPa), which is equal to the low-strain bulk modulus.
  • E an nu - alternative enter E (in MPa) and nu. These are converted to K and G=G1+G2; you can pick G2 (if not zero) to get G1=G-G2
  • UJOption - selects the dilational energy term U(J) and the possible integer values mean:
        0: U(J) = (K/2)((1/2)(J2-1) - ln J) (the default)
        1: U(J) = (K/2)(J-1)2
        2: U(J) = (K/2)(ln J)2
  • alpha - the thermal expansion coefficient (in ppm/C)
  • <IdealRubber/> - this command (with no value) makes the material an ideal rubber elastic material.
  • (other) - see the properties common to all materials.

History Data for Mooney-Rivlin Materials

  1. The volumetric ratio or J = V/V0 (i.e., the determinant of the deformation gradient)

Neo-Hookean Material (Type=28)

The material type is a large-deformation, isotropic, hyperelastic material. The material properties are

  • K, G, Lame, E, and nu - these are the low-strain elastic properties for the material. You must enter K and G, E and nu, or Lame and G (moduli in MPa).
  • UJOption for an integer to select the an energy term U(J) and the possible options are:
        0: U(J) = (Lame/2)((1/2)(J2-1) - ln J) (the default)
        1: U(J) = (Lame/2)(J-1)2
        2: U(J) = (Lame/2)(ln J)2
  • alpha for thermal expansion coefficient in ppm/degree C.
  • (other) - see the properties common to all materials.

History Data for Neo-Hookean Material

  1. The volumetric ratio or J = V/V0 (i.e., the determinant of the deformation gradient)

Isotropic, Elastic-Plastic Material (Type=9)

This material is small-strain, isotropic material with plasticity. The new properties are:

  • Hardening - this property selects the hardening law. It should be followed by entering all yielding properties required by the hardening law within the material's definition block.
  • (other) - see Linear Elastic, Isotopic material for all other properties.

History Data for Isotropic, Elastic-Plastic Material

The material has no history variables, but all hardening law will have one or more history variables.

Bistable, Isotropic Elastic (Type=10)

This material has two small-strain, isotopic states. You need to enter properties of the initial and deformed states and pick rule for state transformations:

  • K0 and Kd - bulk modulus of initial state and transformed state (in MPa).
  • G0 and Gd - shear modulus of initial state and transformed state (in MPa).
  • alpha0 and alphad - thermal expansion coefficient of initial state and transformed state (in ppm/degree C).
  • beta0 and betad - concentration expansion coefficient of initial state and transformed state (in strain/(wt fraction)).
  • D0 and Dd - diffusion constant of initial state and transformed state (in mm2/sec).
  • kCond0 and kCondd - thermal conductivity of initial state and transformed state (in W/(m-K)).
  • transition - set state transition to be by a dilation rule (if 1), a distortion rule (if 2), or Von Mises stress rule (if 3) (the default is dilation).
  • critical - critical value for the transition. The value is volumetric or deviatoric strain (in % for dilation or distortion rule) or stress (in MPa for Von Mises stress rule).
  • DeltaVOffset - offset volumetric strain in the transformed state. This property only applies for dilation rule (see above); it is ignored for distortion rule.
  • <reversible/> - this empty tag makes the transition reversible (the default is reversible).
  • <irreversible/> - this empty element makes the transition irreversible (the default is reversible).
  • (other) - see the properties common to all materials.

History Data for Bistable, Isotropic Elastic Materials

  1. Will be 0 for the initial state and 1 for the deformed state after a transition.

Rigid Material (Type=11)

The material is actually two materials in one, depending on the value for its SetDirection property. If SetDirection is 0 through 7, the material is not really a material. Instead material points with such a material will assign grid-based boundary conditions that translate with the particle. If SetDirection is 8, the material will correspond to actual material points that interact with other material points only through multimaterial contact laws. The material properties are:

  • SetDirection - values 0 through 7 (binary 0 through 0x111) create dynamic velocity boundary conditions in the x, y, and/or z axis according to three bits in the binary value (e.g.,1 or binary 0x001 to set velocity in the x direction). A setting of 8 creates rigid particles that can implement multimaterial contact contact.
  • <SetTemperature/> - this empty tag means to create dynamic temperature boundary conditions (it requires SetDirection 0 to 7).
  • <SetConcentration/> - this empty tag means to create dynamic concentration boundary conditions (it requires SetDirection 0 to 7).
  • SettingFunction, SettingFunction2, or SettingFunction3 - by providing user-defined functions of time, position, and time step, the rigid particle velocity setting can vary. The functions should evaluate to the velocity in mm/s. For SetDirection 1 to 7 the functions must be given in order and apply to the set velocities in order. For SetDirection of 8, the three functions are for setting x, y, and z directions and zero to three can be used.
  • ValueFunction - by entering user-defined functions of time, time step, and position, the rigid particle temperature and concentration can vary. The function should evaluate to the temperature or concentration potential.

Hill Plastic, Orthotropic Material (Type=15)

This material is a small-strain, elastic-plastic material based on an orthotropic material but has plasticity determined by a built-in, anisotropic criterion for yielding:

[ F(σyy - σzz)2 + G(σxx - σzz)2 + H(σyy - σxx)2 + 2Lτyz2 + 2Mτxz2 + 2Nτxy2 ]1/2 - (1 + Kεpn)

where K and n are dimensionless hardening law properties and F through N are determined from yield stress in each material direction. The material properties are:

  • yldxx - yield stress from uniaxial tension test the material's x direction (in MPa)
  • yldyy - yield stress from uniaxial tension test the material's y direction (in MPa)
  • yldzz - yield stress from uniaxial tension test the material's z direction (in MPa)
  • yldxy - yield stress from pure shear test in the material's x-y plane (in MPa)
  • yldyz - yield stress from pure shear test in the material's y-z plane (in MPa)
  • yldxz - yield stress from pure shear test in the material's x-z plane (in MPa)
  • khard - hardening term (see above)
  • nhard - exponent on cumulative plastic strain in hardening term (see above)
  • (other) - see Linear Elastic, Orthotropic material for all other properties.

History Data for Hill Plastic, Orthotropic Material

  1. The cumulative equivalent plastic strain defined as the sum of sqrt(2/3)||dεp||.

Mie-Grüneisen Equation of State Materials (Type=17 and Type=25)

These two materials are isotropic materials with plasticity and find pressure using the Mie-Grüneisen equation of state by large deformation methods. The Type=17 material uses low-strain, hypoelastic-plastic methods for shear and yielding. The Type=25 material uses large-deformation, hyperelastic-plastic methods for shear and yielding. Both materials can use any hardening law for plasticity. The material properties for both these materials are:

  • C0 - the bulk wave speed (in m/sec).
  • S1 - a parameter in Mie-Grüneisen equation of state (dimensionless).
  • S2 - a parameter in Mie-Grüneisen equation of state (dimensionless).
  • S3 - a parameter in Mie-Grüneisen equation of state (dimensionless).
  • gamma0 - a parameter in Mie-Grüneisen equation of state (dimensionless).
  • G - the shear modulus at zero pressure and the reference temperature (in MPa).
  • Hardening - this property selects the hardening law. It should be followed by entering all yielding properties required by the hardening law within the material's definition block.
  • (other) - see the properties common to all materials. This material requires heat capacity Cv and a StressFreeTemp in K.

In this material, the pressure is related to the volumetric compression strain (x = -ΔV/V0) by the Mie-Grüneisen equation of state. When x>0, the material is in compression and pressure is

P = ρ0C02x(1 - 0.5 γ0 x) / (1 - S1x - S2x2 - S3x3)2 + ρ0γ0e

where e is the internal energy. When x<0, the material is in tension and the pressure changes to

P = ρ0C02x + ρ0γ0e

The first term uses the low-strain bulk modulus, which is ρ0C02.

This equation of state automatically handles thermal expansion (assumning a valid Cv was provided) and therefore you do not need to enter a thermal expansion coefficient.

History Data for Mie-Grüneisen Equation of State Material

All hardening law will have one or more history variables. After that, the Type=17 material has no additional history variables, but the Type=25 material tracks relative volume (J = V/V0) in the first history variable after hardening law history variables.

Anisotropic, Elastic-Plastic, Hyperelastic Material (Type=21)

The material type is a large-strain, anisotropic, hyperelastic material for fabric processing. It is currently in development.

Ideal Gas, as a Hyperelastic Material (Type=22)

The material type is an ideal gas, implemented as a large-deformation, isotropic, hyperelastic material. The possible input properties are:

  • P0 - the reference pressure (in MPa) at reference temperature and reference density.
  • T0 - the reference temperature (in Kelvin)
  • rho - the reference density (in g/cm3) at reference temperature.
  • Cv - if heat capacity is omitted or is less than or equal 1, CV is set to (3/2)R for a monotonic gas. If the entered value is greater than 1, CV is set to (5/2)R for a diatomic gas.
  • (other) - see the properties common to all materials.

Clamped Neo-Hookean Material (Type=29)

The material type is a large-deformation, isotropic, hyperelastic material. It yields when the tensile and/or compression elongation reaches critical values of 1+θs and/or 1-θc. The underlying elastic component of the deformation is a neo-Hookean material. After yielding, the mechanical properties soften in tension or harden in compression according to an entered hardening coefficient. The material properties are

  • CritTens and CritComp - enter critical strains θs and θc for critical strains for tension and compression.
  • xihard - hardening parameter ξ. The neo-Hookean moduli are scaled by exp(ξ(1-JP)) where JP is the plastic volume ratio.
  • Elastic - enter 0 to use neo-Hookean potential energy defined in paper that proposed this material or enter 1 to use Neo-Hookean material potential energy. This potential energy applies to elastic component of the deformation.
  • (other) - enter properties for the underlying Neo-Hookean material (note that the UJOption can only be changed when Elastic is 1; UJOption is always 1 when Elastic is 0).

History Data for Clamped Neo-Hookean Material

  1. The volumetric ratio or J = V/V0 (i.e., the determinant of the deformation gradient)
  2. The plastic volumetric ratio or JP = J/JE where JE is determinant of the elastic deformation gradient)

Isotropic, Hyperelastic-Plastic Material (Type=24)

The material is a large-deformation, isotropic, hyperelastic-plastic material. The elastic stresses use a Neohooken material maodel. The properties are:

  • Hardening - this property selects the hardening law. It should be followed by entering all yielding properties required by the hardening law within the material's definition block.
  • (other) - see Mooney-Rivlin material for all other properties, expect this material does not support (or allow) the G2 or the <IdealRubber/> properties.

History Data for Isotropic Hyperelastic-Plastic Materials

The first history variables are determined by the hardening law. After those variables, the next history variable stores the volumetric strain J = V/V0 (i.e., the determinant of the deformation gradient).

Tait Liquid (Type=27)

This material models a Newtonian fluid as a hyperelastic material. The pressure is found from the Tait equation. The properties are:

  • K - the zero-pressure, bulk modulus (in MPa)
  • viscosity - the liquid viscosity (in cP)
  • alpha - the thermal expansion coefficient (in ppm/K)
  • InitialPressure - set initial pressure to a user-defined function of position that evaluates to a pressure in MPa. When liquids are modeled in gravity, the function can set to ρgh, where h is height of liquid above the position (x, y, z).
  • (other) - see the properties common to all materials.

History Variables for Tait Liquid

  1. Stores the volumetric strain ratio J = V/V0 (i.e., the determinant of the deformation gradient).

Triangular Traction Law Material (Type=12)

The traction law has a triangular shape. There are separate traction laws for opening displacement (mode I) and sliding displacements (mode II):

  • sigmaI and sigmaII - the peak stress or cohesive stress of the traction law (σ) (in MPa).
  • JIc and JIIc - the toughness of the traction law or area under the traction law curve (Jc) (in J/m2)
  • delIc and delIIc - the maximum or critical opening displacement (δc) at which debonding occurs (in mm)
  • delpkI and delpkII - the relative displacement at the location of the peak in the triangular law (relative to δc and therefore dimensionless from 0 to 1).
  • kIe and kIIe - the initial elastic slope (k) (in MPa/mm).
  • nmix - the exponent used in the mixed-mode failure criterion Enter nmix<=0 to choose infinity, which is decoupled failure criterion (see below).

You must then enter exactly two of the first three properties for each mode (because they are interrelated). You must enter zero or one for the next two properties. If one is provided, the other will be calculated. If neither k nor δpeak are entered, the peak will be located at δpeak = 0.225926299δc.

Failure occurs when

(GI/JIc)nmix + (GII/JIIc)nmix = 1

where GI and GII are areas under mode I and mode II traction laws up to current displacements. If nmix<=0, failure occurs when either mode reaches its critical COD.

Linear Elastic Traction Law Material (Type=13)

This traction law creates a linear elastic traction law that never fails and therefore never releases energy. The only properties are the slopes for normal and shear opening. If either slope is left out, it will be set to zero.

  • kIe and kIIe - the slope of the linear elastic tractional law (k) (in MPa/mm).

To create a linear elastic traction law that fails at some critical cod, use a triangular traction law material instead.

Cubic Traction Law Material (Type=14)

The traction law uses a cubic function:

σ = (27/4) σmax (δ/δc)(1-(δ/δc))2.

that was designed to have zero slope at the critical COD. There are separate traction laws for opening displacement (mode I) and sliding displacements (mode II):

  • sigmaI and sigmaII - the peak stress or cohesive stress of the traction law (σ) (in MPa).
  • JIc and JIIc - the toughness of the traction law or area under the traction law curve (Jc) (in J/m2).
  • delIc and delIIc - the maximum or critical opening displacement (δc) at which debonding occurs (in mm).
  • nmix - the exponent used in the mixed-mode failure criterion. Enter nmix<=0 to choose infinity, which is decoupled failure criterion.

You must then enter exactly two of the first three properties for each mode. The failure criterion under possible mixed-mode conditions is identical to the one used in the triangular traction law.

Trilinear Traction Law Material (Type=20)

This traction law is a piecewise-linear law with two break points.There are separate traction laws for opening displacement (mode I) and sliding displacements (mode II):

  • sigmaI, sigmaII, JIc, JIIc, delIc, delIIc, kIe, kIIe, delpkI, delpkII, and nmix - these properties are defined in the Triangular Traction Law and they all mean the same thing for this traction law, except that they now apply to the first break point in the trilinear law at (δ1,σ1).
  • sigmaI2 and sigmaII2 - the stress at the second break point in the traction traction law (σ2) (in MPa).
  • delpkI2 and delpkII2 - the relative displacement at the location of the second peak in the trilinear law (relative to δc and therefore dimensionless from 0 to 1).

Excluding nmix, you must enter exactly 5 of the seven properties for each mode. Furthermore, if one of the 5 is kIe (or kIIe for mode II), then either sigmaI or delpkI (but not both) must be specified as well.

The failure under mixed-mode loading uses the same criterion explained above.

CoupledSawTooth Traction Law Material (Type=23)

This traction law also has a triangular shape except that cod and traction are now effective terms that are related to normal and tangential components of traction and displacement:

Traction = Teff where Teff2 = Tn2 + Tt2
cod = δeff where δeff2 = δn2 + δt2

The properties are:

  • sigmaI, JIc, delIc, kIe, and delpkI - these properties are defined in the Triangular Traction Law and they all mean the same thing for this traction law, except they apply now to effective traction and critical effective cod rather than mode I traction and critical cod. Mode II parameters and nmix are ignored.

Pressure Traction Law Material (Type=26)

This traction law creates a constant stress that is normal to the crack surface. It models a pressure loaded crack. The stress can be entered in one of two ways:

  • stress - enter constant normal stress; use negative stress for a pressure loaded crack (in MPa).
  • function - Alternatively, the stress can be entered as a user-defined function of time (in MPa).
  • minCOD - specify a minimum, normal, crack opening displacement below which no pressure is applied (in mm). It is ignored if it is negative (default is -1)

Material Properties Common For All Material Types

There are some material properties that are common to all material types, although not all materials will use them all. The categories of common properties are:

  • Basic Properties
  • Fracture Toughness Properties
  • Crack Propagation Properties
  • Contact Properties
  • Artificial Viscosity

Basic Properties

These are basic material properties

  • rho - the density (in g/cm3)
  • Cv - the constant strain heat capacity (in J/(kg-K)). You do not need to enter a constant pressure heat capacity because material models calculate it from Cv and thermodynamic relations for the difference between Cp and Cv.
  • kCond - the thermal conductivity for isotropic materials (in W/(m-K))
  • csat - the saturation concentration entered as a weight fraction
  • beta - the concentration expansion coefficient for isotropic materials (in strain/(wt fraction))
  • D - the diffusion constant for isotropic materials (in mm2/sec)
  • <color red='0.1' green='0.5' blue='0.3' alpha='1.0/> - sets the color of the material with red, green, blue, and alpha values between 0.0 and 1.0.

Fracture Toughness Properties

These properties set material properties that determine the fracture toughness of the material and control various aspects of crack propagation. Note that whether or not the properties are used depends on the selected criterion for crack growth.

  • JIc or JIc - critical mode I or mode II energy release rate (in J/m2).
  • KIc or KIIc - critical mode I or mode II stress intensity factor (in MPa m0.5).
  • KIexp or KIIexp - the exponent p or q in the elliptical criteria for crack growth.
  • delIc or delIIc - critical crack opening displacements in mode I and mode II.
  • initTime - time to start crack growth (in ms).
  • speed - define the constant crack speed (in m/sec).
  • maxLength - define the maximum length of the crack (in mm). The crack growth will stop when this length is reached.
  • nmix - a parameter used in some mixed-mode failure laws.

Crack Propagation Properties

The criterion, direction, and traction properties (and the analogous alternate propagation properties) are set with following XML commands (instead of standard property commands):

  • <Propagate criterion='1' direction='0' traction='1'/> - lets each material have a different primary crack propagation criterion and/or directions than the ones defined in the MPM Header crack settings (and set as described there).
  • <AltPropagate criterion='7' direction='4' traction='1'/> - same as Propagate command except for the secondary propagation criterion.
  • xGrow and yGrow - specify a unit vector for a constant crack growth direction. They only apply for crack propagation with criterion 2 and then only when that criterion selects the default propagation direction.
  • constantTip - set to 0 or 1 where 1 means the crack tip will always uses its initial material even if the crack is in or moves to another material

Contact Properties

These commands allow custom contact properties for each material pair in multimaterial mode MPM.

  • <Friction mat='(num)' Dt='(Dt)' Dn='(Dnt)' Dnc='(Dnc)'>(frict)</Friction> - set custom material-material contact properties between current material and the material in the mat attribute. It is set as described in Multimaterial Mode MPM and overrides that setting with custom settings.
  • shareMatField - set to another material number (by number only) to share material velocity field with that material. Two (or more) materials sharing the same velocity field will move with perfect contact (i.e., in a single velocity field) among themselves, but can interact with other materials in different velocity fields by contact or interface laws. Materials sharing velocity fields must be compatible (e.g., all rigid or all nonrigid) and the material in (matID) must not be sharing it's field.

Artificial Viscosity

Artificial viscosity adds a pressure, Q, related to velocity gradient on the particle, but only when it is compressing. The equation is:

Q = Δx*|Dkk|*(A1*C + A2*Δx*|Dkk|)

where Δx is the cell size of the mesh, |Dkk| is the relative volume change rate, C is the bulk wave speed in the material, and A1 and A2 are adjustable constants.

  • <ArtificialVisc/> - the presence of this empty element activates artificial viscosity.
  • avA1 - the A1 parameter in artificial viscosity (dimensionless, default 0.2)
  • avA2 - the A2 parameter in artificial viscosity (dimensionless, default 2.0)

These parameters only work in materials that support artifical viscosity; an error will result if used in other materials.
