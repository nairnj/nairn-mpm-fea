The nairn-mpm-fea Project

Introduction

This documentation is for nairn-mpm-fea computational mechanics project.

OSParticulas (or NairnMPM) and NairnFEA Code Engines

The calculation engines OSParticulas (or NairnMPM) and NairnFEA do material point method (MPM) and finite element analysis (FEA) calculations, respectively. They are object-oriented, C++, platform-independent code engines.

Visualization Options

The java application NairnFEAMPMViz included in the project provides platform-independent tools for running and visualizing calculations. For Mac-specific calculations and visualization (with more features than NairnFEAMPMViz), see the NairnFEAMPM web site.

Documentation

This documentation is in the process of be converted to a web-based wiki called OSUPDocs.

Since you are reading this documentation, you must have already checked out a copy of the nairn-mpm-fea package (the folders in the project are explained here). This documentation explains how to compile all tools, create input files for calculations, run calculations, and gives the options for visualizing and interpreting the results. The code can be run on multiple platforms including MacOS X, Windows, and Linux machines.

  • Getting Starting - compiling the source code.
  • Running the Code - running the code engines.
  • Visualization - options for visualizing the results.
  • XML Input File Format - definition of the XML file commands needed to set up FEA or MPM calculations.
  • Output Results Format - format for output files of FEA and MPM results
  • References - selected references related to use and development of these code engines
